# User Guide

## Быстрый старт
1. `npm start`.
2. Выбери проект на Дашборде (или он откроется автоматически).
3. Кликни на карточку проекта или нажми "Open Workspace".

## Хоткеи (работают на любой раскладке)

### Глобальные
- `Cmd + \`: Проводник файлов (File Explorer).
- `Esc`: Закрыть превью файла.

### Контекстно-зависимые
**В Workspace (когда открыт терминал):**
- `Cmd + T`: Новая вкладка в текущем проекте.
- `Cmd + W`: Закрыть активную вкладку (с подтверждением если есть процессы).

**На Dashboard:**
- `Cmd + W`: Закрыть активный проект (chip в title bar).

### Дополнительные
- `Cmd + B`: Сайдбар (Заметки/AI/Действия).
- **Double Click по табу**: Переименовать вкладку.
- **Наведение на project chip**: Кнопка × для закрытия проекта.

## Визуальные индикаторы
- **Активный проект** (в title bar): Яркий синий цвет, тень, белый индикатор.
- **Активная вкладка**: Синяя полоска сверху, белый текст.
- **Активные процессы**: При закрытии вкладки/проекта появится подтверждение.

## Фишки
- **Gemini Search:** Выдели текст в терминале -> Правая кнопка -> "Search Reddit with Gemini".
- **Quick Actions:** В Сайдбар вынесены кнопки для частых команд. Их можно менять на Дашборде в разделе Settings.
- **File Preview:** Кликни на файл в проводнике (`Cmd + \`), чтобы быстро глянуть код.
- **Project Cards:** Карточки проектов с описанием, путем и количеством активных табов. Наведи мышку на ℹ️ чтобы увидеть описание.
