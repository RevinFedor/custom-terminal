# Feature: Settings & Prompts Management

## Intro
Модальное окно настроек (`Cmd+,`) является единым центром управления конфигурацией приложения, включая шрифты, цвета, горячие клавиши и параметры AI-агентов.

## User Flow: Настройка AI
1. Пользователь переходит на вкладку **"AI"**.
2. В левой колонке редактирует системные промпты (Claude Default, Research, Compact).
3. В правой колонке управляет пользовательскими промптами для контекстного меню.
4. **Behavior:** Изменения сохраняются мгновенно при выходе из поля ввода (**onBlur**). Кнопка "Done" отсутствует для упрощения UX. См. `knowledge/ui-ux-stability.md`.

## User Flow: Горячие клавиши
1. При первом запуске или нажатии `Cmd+,` открывается вкладка **"Горячие клавиши"**.
2. Пользователь видит список актуальных шорткатов на русском языке (два столбца).
3. **Behavior:** Список фильтруется, отображая только самые важные команды (управление табами, сайдбаром, редактором).

## Behavior Specs
- **Dynamic Width:** Ширина модального окна автоматически увеличивается до 900px при переходе на вкладку AI для комфортной работы с двумя колонками.
- **Persistence:** Последняя выбранная вкладка сохраняется в рамках сессии. При повторном открытии пользователь возвращается туда, где остановился.
- **Auto-save:** Механизм `onBlur` использует проверку `relatedTarget`, чтобы не закрывать режим редактирования при клике на служебные кнопки (например, Reset) внутри того же блока.

## Code Map
- `SettingsModal.tsx`: Главный компонент модального окна.
- `useUIStore.ts`: Хранение настроек `chatSettings`, `docPrompt` и пользовательских промптов.
- `database.js`: Синхронизация `app_state` (например, `claudeDefaultPrompt`) с SQLite.
