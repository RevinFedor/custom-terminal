# Feature: Global Settings

## Intro
Глобальное модальное окно для управления внешним видом и системными параметрами терминала. Разделено на логические секции через табы.

## Behavior Specs
- **Хоткей:** `Cmd + ,` (запятая) открывает настройки.
- **Вкладка "Fonts" (Шрифты):**
    - Управление размером шрифта терминала (8-24px).
    - Управление размером вкладок воркспейса (Main + Utils).
    - Управление размером чипов проектов в заголовке окна.
    - Изменения применяются мгновенно ко всем открытым инстансам.
- **Вкладка "Colors" (Цвета):**
    - Справочник используемых цветов для вкладок (разбивка на системные и пользовательские).
- **Persistence:** Все настройки сохраняются в `localStorage` и восстанавливаются при запуске.

## Code Map
- **UI:** `src/renderer/components/UI/SettingsModal.tsx`.
- **Компоненты:** `FontSizeSlider` — переиспользуемый компонент для управления числовыми значениями через range-input.
- **Store:** `src/renderer/store/useUIStore.ts` — централизованное хранилище настроек интерфейса.
- **Event Loop:** `src/renderer/App.tsx` перехватывает `Cmd+/-` для быстрого изменения шрифта терминала. См. `knowledge/fix-zoom-override.md`.
