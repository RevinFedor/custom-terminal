# Features: Product Logic

## Карта Модулей
- **Управление Табами:** `features/tabs.md` — Цвета, Drag & Drop, Smart Restart (↻), Restart Zone, Smart New Tab (в текущем CWD).
- **Research (AI Чат):** `features/research.md` — Интеграция Gemini, режимы Research и Compact, работа с выделением.
- **AI Промпты и Команды:** `features/prompts.md` — Быстрые команды терминала и шаблоны для AI.
- **Глобальные Настройки:** `features/settings.md` — Шрифты, цвета и системные настройки.
- **AI Сессии:** `features/ai-sessions.md` — Автоматизация, восстановление сессий, прямое внедрение чекпоинтов.
- **Claude Сессии:** `features/claude-sessions.md` — Управление через InfoPanel, Claude Runner, Smart Fork, Interrupted Sessions.
- **Update Docs:** `features/docs-update.md` — Синхронизация Claude и Gemini для документации.
- **Workspace:** `features/workspace.md` — Проводник (Overlay), превью файлов, Gemini Search, заметки проекта.
