# Features: Product Logic

## Карта Модулей
- **Управление Табами:** `features/tabs.md` — Цвета, Drag & Drop, Main/Utils зоны, сохранение визуальной истории (Serialization).
- **Research (AI Чат):** `features/research.md` — Интеграция Gemini, система чатов, работа с выделением.
- **AI Промпты и Команды:** `features/prompts.md` — Быстрые команды терминала и шаблоны для AI.
- **Глобальные Настройки:** `features/settings.md` — Шрифты, цвета и системные настройки.
- **AI Сессии:** `features/ai-sessions.md` — Автоматизация, восстановление сессий, прямое внедрение чекпоинтов.
- **Update Docs:** `features/docs-update.md` — Синхронизация Claude и Gemini для документации.
- **Workspace:** `features/workspace.md` — Проводник, превью файлов, Gemini Search, Empty States.

## База Знаний (Knowledge)
- **Стабильность UI:** `knowledge/fix-ui-stability.md` — Jitter, Flickering, Fonts, Zoom.
- **Рендеринг:** `knowledge/fix-markdown-hydration.md` — Решение проблем с Markdown и блочными элементами.
- **AI Автоматизация:** `knowledge/fix-ai-sessions.md` — State Detection, Cursor detection, Trojan Horse.
- **UX и Ввод:** `knowledge/fix-input-ux.md` — Enter key, Modal dialogs.
- **Данные:** `knowledge/fix-data-persistence.md` — Persistence, Dynamic Styles.
