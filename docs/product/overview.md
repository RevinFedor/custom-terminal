# Noted Terminal

Интеллектуальная оболочка над Claude Code и Gemini CLI. Это не просто терминал, а система глубокой интеграции, которая превращает консольных агентов в структурированную среду разработки с визуальной навигацией и автоматизацией.

### Почему это работает лучше обычного терминала

**Событийное управление без принудительных задержек** *гарантированная доставка команд*
    - система не угадывает готовность интерфейса по таймерам, а ждёт маркер синхронизации отрисовки (`OSC 2026`)
    - это позволяет выполнять сложные цепочки (смена модели → откат → вставка контекста) со 100% надёжностью
    - механизм безопасной вставки (`safePasteAndSubmit`) верифицирует каждый фрагмент данных через поток терминала
    - результат: ноль «проглоченных» символов и отсутствие конфликтов при автоматизации команд

**Обратная трассировка и разбор логов** *честная история без мусора*
    - приложение реконструирует дерево диалога, отсекая «мёртвые ветки» после отмен и случайных нажатий
    - это даёт разработчику чистую ленту истории, соответствующую реальному состоянию контекста нейросети
    - автоматическое связывание через «мосты» бесшовно склеивает сессии после сброса контекста или сжатия истории
    - инкрементальное чтение обрабатывает огромные файлы логов (JSONL) асинхронно, исключая зависания интерфейса

**Визуальная лента истории** *мгновенный прыжок к любому сообщению*
    - вертикальная панель справа превращает терминальный буфер в интерактивную карту проекта
    - клик по точке навигирует терминал точно к строке вопроса через специальные маркеры в потоке данных
    - индикация «зон досягаемости» сразу показывает, какой контент ещё есть в памяти терминала, а какой уже удалён
    - группировка работы суб-агентов скрывает десятки технических шагов под одну компактную ноду

### Управление и оптимизация контекста

**Умный откат и правка сообщений** *экономия токенов без потери смысла*
    - при откате истории Gemini автоматически создаёт сжатую сводку удаляемых сообщений
    - этот «смысловой остаток» возвращается в чат, сохраняя важные факты при очистке контекста
    - функция правки диапазона позволяет вырезать и пережать любую часть диалога прямо в файле сессии
    - это превращает работу с длинными задачами в управляемый процесс менеджмента памяти нейросети

**Координация агентов: Gemini → Claude** *терминал как диспетчер*
    - основной агент (Gemini) может делегировать задачи помощнику (Claude) напрямую через текстовый паттерн
    - приложение выступает «умным мостом»: перехватывает инструкции, следит за занятостью и возвращает результат
    - многоуровневая защита предотвращает ложные срабатывания на индикаторах загрузки и перерисовках экрана
    - суб-агенты работают в скрытых вкладках, не занимая основное пространство, но оставаясь под контролем

**Конвейер обновления документации** *автоматизация актуальности знаний*
    - двухстадийный процесс (анализ → применение) для синхронизации документации с изменениями в коде
    - автоматическая подстановка файлов базы знаний в каждый запрос для сохранения контекста проекта
    - визуальные маркеры на ленте истории отмечают моменты, когда нейросеть вносила правки в файлы знаний

### Производительность и стабильность

**Стабильная отрисовка интерфейса** *плавность при любых объёмах вывода*
    - специальная настройка окружения заставляет агентов оптимизировать вывод под современные стандарты
    - защита кадров синхронизации убирает мерцание и прыжки экрана при интенсивном обновлении текста
    - аппаратное ускорение и механизмы плавных переходов обеспечивают мгновенный отклик даже в очень длинных чатах
    - кастомный индикатор прокрутки не мешает изменению размеров панелей, сохраняя каждый пиксель рабочего пространства
