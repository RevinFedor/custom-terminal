# Main Features: User Flows

Этот файл описывает основные пользовательские сценарии (Flows) и связывает их с детальной документацией.

## 1. Flow: Работа с проектами и навигация
Пользователь управляет своими рабочими пространствами через верхнюю панель управления проектами (Project Tabs).
- **Действие:** Клик по вкладке проекта переключает весь контекст (терминалы, проводник, заметки).
- **Действие:** Правый клик по проекту открывает контекстное меню для закрытия, а нажатие колесиком мыши закрывает проект мгновенно.
- **Особенности:** Используется стиль "portfolio" (активный проект выделен белым). При закрытии проверяется наличие активных процессов.
- **Подробнее:** [`features/workspace.md`](features/workspace.md)

## 2. Flow: Умное управление терминалом
Пользователь взаимодействует с вкладками терминала, которые ведут себя адаптивно.
- **Действие:** Пользователь вводит `npm run dev`. Таб автоматически окрашивается в зеленый и переименовывается в `run-dev`.
- **Действие:** Пользователь запускает `claude`. Таб становится оранжевым, активируется Sniper Watcher.
- **Действие:** Пользователь запускает `gemini`. Таб становится синим, активируется Gemini Sniper. Сессия захватывается после первого сообщения.
- **Действие:** Форк сессии (⑂). Для Claude это восстановление по UUID, для Gemini — True Fork (создание физической копии состояния).
- **Подробнее:** [`features/tabs.md`](features/tabs.md), [`features/claude-sessions.md`](features/claude-sessions.md), [`features/gemini-sessions.md`](features/gemini-sessions.md)

## 3. Flow: AI Исследование (Research)
Пользователь получает помощь AI, не выходя из контекста терминала.
- **Действие:** Выделение ошибки в терминале -> ПКМ -> "Research Selection".
- **Результат:** Открывается правая панель (Research Panel), запускается поиск Gemini с использованием [State-driven триггера](knowledge/fix-gemini-search.md).
- **Взаимодействие:** Пользователь может скопировать ответ, нажать "Retry" для перегенерации или удалить сообщение из истории.
- **Подробнее:** [`features/research.md`](features/research.md)

## 4. Flow: Синхронизация Документации
Пользователь передает контекст из одной нейросети в другую для обновления документов.
- **Действие:** Нажатие кнопки "Update Docs" в панели экшенов.
- **Процесс:** Приложение отправляет `/export` в терминал Клода, дожидается появления файла через [FS Polling](knowledge/fix-ai-sessions.md) и скармливает его Gemini в новом сервисном терминале.
- **Подробнее:** [`features/docs-update.md`](features/docs-update.md)

## 5. Flow: Безопасное восстановление (Persistence)
Приложение гарантирует, что работа не будет потеряна при перезагрузке или сбое.
- **Действие:** Пользователь закрывает приложение. Все табы, их цвета, CWD и ID сессий сохраняются в SQLite.
- **Сценарий Сбоя:** Если сессия была прервана, при следующем открытии появится [Interrupted Overlay](features/claude-sessions.md), предлагающий восстановить контекст одной кнопкой.
- **Подробнее:** [`features/ai-sessions.md`](features/ai-sessions.md), [`knowledge/fix-data-persistence.md`](knowledge/fix-data-persistence.md)

## 6. Flow: Управление заметками (Notes)
Пользователь ведет проектную документацию и личные заметки, привязанные к контексту проекта.
- **Действие:** Запись быстрых мыслей в боковую панель Info.
- **Действие:** Нажатие `Cmd+E` разворачивает полноэкранный Markdown-редактор поверх терминала.
- **Особенности:** Автосохранение при закрытии, поддержка Markdown, мгновенная синхронизация между панелью и редактором.
- **Подробнее:** [`features/notes.md`](features/notes.md)

## 7. Flow: Машина времени (Time Machine)
Пользователь может откатить состояние Gemini сессии к любому предыдущему сообщению.
- **Действие:** В InfoPanel раскрывается секция Time Machine.
- **Процесс:** Пользователь видит список всех своих сообщений (Turns) с превью.
- **Результат:** Нажатие на ↺ удаляет "будущее" сессии, подменяет файл сессии снепшотом и перезапускает терминал в той же точке.
- **Подробнее:** [`features/time-machine.md`](features/time-machine.md)

## Технические "фундаменты" (Knowledge Base)
Все флоу опираются на критические решения:
- **Форк:** [True Fork для Gemini](knowledge/fix-gemini-true-fork.md) — почему просто resume не работает.
- **Стабильность:** [Устранение мерцания AI UI](knowledge/fix-ai-ui-flicker.md) через прямую инъекцию ID.
- **Отрисовка:** [Canvas вместо WebGL](knowledge/fix-ui-stability.md) для стабильности.
- **Пути:** [Реактивный OSC 7](knowledge/fact-osc7-cwd.md) для мгновенного отслеживания CWD.
- **Слои:** [Layering Pattern](knowledge/fix-layering-pattern.md) для отрисовки UI поверх терминала.
- **Ввод:** [Safe Write](knowledge/fix-pty-buffer-overflow.md) для вставки огромных промптов без обрывов.
