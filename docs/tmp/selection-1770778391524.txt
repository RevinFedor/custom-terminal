# Favorites Feature + History Collapse + ProjectHome Context Menu

## Context

## Constraints
- `data-persistence.md`: Persist через SQLite + IPC (паттерн `tab_history`)
- `ui-ux-stability.md`: Portal buttons → `e.stopPropagation()`, no `useCallback` для часто меняющегося стейта
- `rendering-styles.md`: Inline styles для динамических цветов
- `ui-ux-stability.md#11.1`: ProjectHome — child[2] с absolute z-50, не сдвигает индексы

## Plan

### 1. Database: таблица `favorites` + методы
**File: `src/main/database.js`**

- Создать таблицу `favorites` в `initTables()` (после `tab_history`):
  ```sql
  CREATE TABLE IF NOT EXISTS favorites (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    project_id TEXT NOT NULL,
    name TEXT, cwd TEXT, color TEXT, notes TEXT,
    command_type TEXT, tab_type TEXT DEFAULT 'terminal',
    url TEXT, claude_session_id TEXT, gemini_session_id TEXT,
    created_at INTEGER DEFAULT (strftime('%s', 'now')),
    FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
  )
  ```
- Индекс: `idx_favorites_project`
- Методы: `addFavorite(projectId, tab)`, `getFavorites(projectId)`, `deleteFavorite(id)`

### 2. IPC handlers
**File: `src/main/main.js`**

Добавить 3 handler-а (рядом с tab_history handlers):
- `project:add-favorite` → `db.addFavorite()`
- `project:get-favorites` → `db.getFavorites()`
- `project:delete-favorite` → `db.deleteFavorite()`

### 3. TabBar: "Add to Favorites" в контекстном меню
**File: `src/renderer/components/Workspace/TabBar.tsx`**

- В контекстное меню (после "Copy ID/Path", ~line 1679) добавить пункт "Add to Favorites"
- При клике: `ipcRenderer.invoke('project:add-favorite', { projectId, tab: { name, cwd, color, notes, commandType, tabType, url, claudeSessionId, geminiSessionId } })`
- Toast: `showToast('"tab.name" added to favorites', 'success')`
- `e.stopPropagation()` на onClick

### 4. Terminal context menu: "Add to Favorites"
**Files: `src/main/main.js`, `src/renderer/components/Workspace/Terminal.tsx`, `src/renderer/App.tsx`**

- **App.tsx** (~line 724, `handleContextMenuCommand`): обработать `'add-to-favorites-from-terminal'` — прочитать таб из store, вызвать IPC `project:add-favorite`, показать toast

### 5. ProjectHome: секция Favorites
**File: `src/renderer/components/Workspace/ProjectHome.tsx`**

### 6. ProjectHome: правый клик → контекстное меню
