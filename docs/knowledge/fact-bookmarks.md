# Feature: Bookmarks (Reserved Directories)

## Intro
Система «Закладок» позволяет пользователю резервировать часто используемые директории и быстро создавать на их основе новые рабочие среды (проекты).

## User Flow
1. **Добавление:** На Dashboard пользователь нажимает «Add Directory» и выбирает папку.
2. **Просмотр:** В секции «Reserved Directories» появляется карточка с именем папки.
3. **Описание:** Пользователь может навести на иконку «?» и увидеть описание папки (Genie Tooltip).
4. **Создание проекта:**
   - При наведении на карточку появляется зеленый оверлей «Add Project».
   - Клик по карточке создает **новый инстанс** проекта из этой папки.
   - Если такой проект уже открыт, создается копия с суффиксом (например, `hh-tool-1`). См. `knowledge/fact-data-persistence.md`.

## Behavior Specs
- **Genie Tooltip:** Анимация появления из точки «?» с использованием `origin-right/left` и масштабирования. Направление выбирается автоматически в зависимости от свободного места на экране.
- **Manual Bounds Check:** Предотвращение «залипания» фокуса при взаимодействии с меню. См. `knowledge/fact-react-patterns.md`.
- **Sync:** Закладки хранятся в таблице `bookmarks` SQLite.

## Code Map
- `BookmarkCard.tsx`: Компонент карточки с логикой hover и создания проекта.
- `useBookmarksStore.ts`: Управление состоянием закладок.
- `database.js`: SQL запросы для CRUD закладок.
