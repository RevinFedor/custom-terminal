# Feature: Terminal Search (Cmd+F)

## Intro
Система полнотекстового поиска по буферу терминала, аналогичная поиску в VS Code.

## User Flow
1. Пользователь нажимает `Cmd+F`.
2. В верхнем правом углу терминала появляется панель поиска.
3. При вводе текста совпадения подсвечиваются в буфере.
4. Кнопки `▲`/`▼` (или `Enter`/`Shift+Enter`) позволяют перемещаться между результатами.
5. Нажатие `Esc` закрывает поиск и очищает подсветку.

## Behavior Specs
- **Counter:** Панель отображает текущий индекс совпадения и общее количество (например, "2/15").
- **Proposed API:** Для корректной работы события `onDidChangeResults` используется Proposed API xterm.js (`allowProposedApi: true`).
- **Persistence:** Состояние поиска (текст, результаты) уникально для каждого таба и сбрасывается при закрытии панели.
